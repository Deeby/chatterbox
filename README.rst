---------------
**Chatterbox**는 `카카오톡 플러스친구 자동응답 API <https://github.com/plusfriend/auto_reply>`_를 활용하여 챗봇을 만들 때 사용되는 파이썬 라이브러리입니다. 

Installation
---------------

::

    $ pip install chatterbox.pypi
