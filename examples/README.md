# :rocket: Examples
## [`flask_basic.py`]
기초적인 [chatterbox]의 활용법을 볼 수 있습니다. [Postman]을 통해서 예제가 동작하는 모습을 확인할 수 있습니다.

## [`flask_advance.py`]
[Text type keyboard]의 동작과 주관식 답변을 어떻게 처리하는지 볼 수 있습니다. Chatter의 파라매터인 `memory`, `frequency`, `fallback`의 동작을 확인해볼 수 있습니다.
